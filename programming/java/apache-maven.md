# Apache Maven

Структура файла `$project.pom`.

```xml
<?xml version="1.0" encoding="UTF-8"?>
<project ...>
    <groupId>...</groupId>
    <artifactId>...</artifactId>
    <version>...</version>

    <properties>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
        ...
    </properties>

    <dependencies>
        ...
    </dependencies>
</project>
```
